# Documentación de Desarrollo

Esta carpeta contiene documentación generada durante el desarrollo del proyecto, principalmente relacionada con:
- Análisis de datos CSV
- Verificaciones de importación
- Resolución de problemas
- Proceso de importación de cierres históricos

## 📁 Estructura

### `/analisis/`
Documentos de análisis de estructura de datos CSV y contadores.

### `/verificacion/`
Documentos de verificación de datos importados y cálculos.

### `/importacion/`
Documentos relacionados con el proceso de importación de cierres desde CSV.

## ℹ️ Nota

Estos documentos son principalmente de referencia histórica y no son necesarios para el uso normal del sistema.
Para documentación de usuario y técnica actual, consulta la carpeta principal `docs/`.
